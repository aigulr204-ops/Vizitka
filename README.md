# Сайт-визитка Айгуль

Это одностраничный сайт-визитка для Айгуль, управляющей детского центра Умка в Уфе.

## Как открыть

1. Открой папку:

   `/Users/macbookair/Documents/Codex/aigul-visitka`

2. Дважды нажми на файл:

   `index.html`

Страница откроется в браузере. Ничего устанавливать не нужно.

## Что можно заменить позже

- контакт для связи;
- фото или фирменное изображение;
- точные услуги и направления;
- ссылку на сайт или соцсети детского центра.
